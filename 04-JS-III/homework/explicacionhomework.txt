Arrays: es un tipo de dato, este tiene una estructura en el cual se almacenaran diferentes elementos del mismo tipo o al menos relacionados entre si, estos pueden ser unidimensionales o multidimensionales o sea n*n datos en forma de una matriz. Cada elemento dentro de estos se le asigna una posicion, en programacion las posiciones se empiezan a contar desde 0 en el caso de un arreglo con 5 elementos las posiciones iran desde 0 hasta 4 recorriendo asi todos los elementos una manera mas simple de determinar la ultima posicion de un elemento seria un arreglo con n elementos n-1 seria su ultima posicion.
